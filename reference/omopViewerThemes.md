# List the available configured `OmopViewer` themes

List the available configured `OmopViewer` themes

## Usage

``` r
omopViewerThemes()
```

## Value

A character vector with the available `OmopViewer` themes.

## Examples

``` r
library(OmopViewer)

omopViewerThemes()
#> [1] "darwin"    "default"   "empty"     "ohdsi"     "sad_robot"
```
